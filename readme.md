# E-Challan Dashboard (Streamlit)

This project is a simple dashboard built using Streamlit to analyze daily e-challan data.
It shows total challans, pending challans, disposed challans, and trend over time.

## Features

* KPI cards (Total, Pending, Disposed)
* Challan trend line chart
* Top 10 days with highest challans
* Clean and simple UI

## Tools Used

* Python
* Streamlit
* Pandas
* Matplotlib

## Project Structure

```
app.py
echallan_daily_data.csv
requirements.txt
README.md
```

## How to Run Locally

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the app:

```
streamlit run app.py
```
